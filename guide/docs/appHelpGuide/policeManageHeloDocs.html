# 1.首页

民警管理首页展示的是单位列表，点击右下角按钮可以添加单位。搜索框可以输入年份或者姓名进行搜索。

# 2.民警列表

列表展示是单位中所有的民警，右上角按钮可以添加民警和编辑单位。点击民警可查看详情，点击民警右侧按钮可拨打电话或发送短信。
搜索框可搜索单位内民警。

# 3.民警信息

民警信息展民警信息以及领导信息，点击右上角按钮可删除民警、点击民警信息卡片右侧按钮可修改民警、也可以点击领导信息卡片右侧添加按钮添加领导。

# 4.添加民警信息

* 为必填项,出生日期不能晚于当前日期。

# 5.修改民警信息

修改数据后点击右上角按钮保存修改。

# 6.添加领导

* 为必填项,手机号必须为11位数字。

# 7.领导信息

点击卡片右上角可以修改领导信息，点击右上角按钮可删除领导。

# 8.修改领导信息

修改数据后点击右上角按钮保存修改。

# 9.添加/修改单位

# 9.1添加单位

输入单位名称选择年份后，点击右上角按钮完成添加。


# 9.2修改单位

单位内民警不为空无法删除单位。
