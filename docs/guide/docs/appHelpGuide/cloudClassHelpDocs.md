##### 新建课程
##### 1.点开首页这个蓝色的添加按钮就可以进入新建页面了
![新建](./img/cloudclass/newbuild.png)
##### 2.在新建页面中填写内容，带*号的代表必填项，填写完点击提交按钮(填写中途退出可暂时保存草稿)。提交以后可以在首页中找到自己新建的课程

![新建课程](./img/cloudclass/buildcourse.png)

![所有课程](./img/cloudclass/courselist.png)
