# 1.首页

首页左侧展示部门列表右侧展示部门内员工列表，右侧内容会根据左侧点击的部门产生变化。点击右下角按钮可添加部门，点击右上角加号按钮可添加民警,点击部门右侧【编辑按钮】编辑部门。

# 2.民警信息

民警信息展民警详情，点击右上角按钮可删除民警、点击民警信息卡片右侧按钮可修改民警

# 3.添加民警信息

* 为必填项，手机号必须为11位数字。

# 4.修改民警信息

修改数据后点击右上角按钮保存修改。

# 5.添加/修改部门

# 5.1添加部门

* 为必填项，点击右上角按钮完成添加。


# 5.2修改部门

部门内民警不为空无法删除部门。
