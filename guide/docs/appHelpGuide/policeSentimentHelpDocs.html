# 1. 应用首页

进入首页，顶部栏图标依次是【更多】以及【退出应用】。

点击【右下图标】进入新建警情，首页由上往下主要分为第一块是警情数据的一个统计总和。

其次是三个页面跳转入口：

  • 我的任务
  • 已派发警情
  • 我的上报

警情共分为三种状态（未派发、处理中、处理完毕）

  • 未派发：指该警情上报了但是后台管理还没有将它交给哪位警官来处理。
  • 处理中：指该警情已经派发给某位警官，但是还没有处理完毕。
  • 处理完毕：指某位警官已经处理完该警情。

再往下分别是一个按警情类型显示总数据的饼图和一个警情趋势统计的折线图，记录着当前一周的警情上报的情况。

# 1.1 我的任务

上级派发给你的警情任务信息都在这里，当内容过多时，你也可以选择【警情类型】、【报警时间】、【处理情况】进行精确查询。

# 1.2 已派发警情

进去显示的是某某警情派发给某某派出所的部门列表，如果部门列表过多也可以点击头部进行筛选。

再点击该部门就可以看到该部门相关的警情列表。

# 1.3 我的上报

这里可以看到自己上报过的所有警情。

# 2. 新建

进入新建界面输入内容，填写姓名、选择性别、时间、电话等信息，输入完内容可点击【右上角】进行提交。

# 3. 详情

警情详情分为报警信息和接警信息，报警信息是报警人填写的一些相关信息。

接警信息是由后台管理派发给某位民警将其填写并进行【提交】的。（带有*号为必填项）

# 4. 更多

点击【左上角】可以返回首页，内容分为应用版本和开发公司相关信息。
