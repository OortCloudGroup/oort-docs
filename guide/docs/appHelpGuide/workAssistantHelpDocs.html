# 1. 应用首页

进入首页，顶部栏图标依次是【搜索】、【更多】以及【退出应用】。

首页有个日历组件，当点击不同的时间底下就会显示不同的内容，点击【右下图标】进入新建界面。

# 2. 新建

可以选择任意类型进行创建。(会议、日程、任务、备忘。)

进入新建界面输入内容，选择开始、结束时间，选择对应的人员，上传图片、附件，输入完内容可点击【右上图标】进行提交。（备忘添加后仅自己可见。）

# 3. 详情

# 3.1 日程详情

点击单选框可以对该日程进行【完成】状态处理，点击右上角发布人也可以对其进行编辑和删除，如果该日程为完成状态就不可以进行编辑了，最底部可对该日程进行【评论】操作。

# 3.2 会议详情

点击单选框可以对该会议进行【已结束】状态处理，点击右上角发布人也可以对其进行编辑和取消会议处理，如果该会议为结束状态就不可以进行编辑了，最底部可对该会议进行【评论】操作。

# 3.3 任务详情

点击单选框可以对该任务进行【完成】状态处理，点击右上角发布人也可以对其进行编辑和关闭任务处理，如果该任务为完成状态就不可以进行编辑了，最底部可对该任务进行【评论】操作。

# 3.4 备忘详情

进入备忘详情页面点击右上角发布人也可以对其进行编辑和删除处理。（备忘添加后仅自己可见。）

# 4. 更多

点击【左上角】可以返回首页，内容分为应用版本和开发公司相关信息。
