# 1. 应用首页

进入首页，顶部栏图标依次是【更多】以及【退出应用】，点击【右下图标】进行新建任务。

首页分为上下结构，上边目录分为任务列表和岗位设置，当你点击不同的目录，下边就会展示不同的内容列表。

下边内容左边是一个任务列表，任务分为三种状态（未开始、进行中、已过期），我们也可以对任务进行【编辑】、查看【详情】。

右边分为值班和勤务两大类型，值班类型可点击部门展示对应的值班人，勤务类型展示出所有与该勤务相关的人员。

# 2. 岗位设置

我们可以点击【右下图标】进行新建岗位。

进入新建岗位界面后，需要填写勤务名称，选择类型和时间范围、备注后可点击【右下角】进行保存。

勤务类型的岗位，按住一秒以上可进行删除和编辑。

点击删除弹出提示框。

点击编辑进入编辑勤务页面。

# 3. 添加任务

进入添加任务界面输入任务名称，选择对应的时间范围后可在【右下角】选择【保存】或【排班编辑】。

# 4. 编辑任务

如果点击保存提示成功后将会返回首页，我们就可以在左边的任务列表看见我们刚刚添加的 “隐秘任务” 了，如果我们想改个名字就可以对它进行编辑。

如果点击了排班编辑将会跳转到排班编辑页面。

排班编辑页面左边是一个部门列表，点击不同的部门右边将会展示部门对应的人员列表，还可以给每个人员添加对应的值班和勤务岗位。（如果勾选了再点一次取消即可）

# 5. 任务详情

在首页点击对应任务的下的【详情】即可进入任务详情页。

这里也是跟排班编辑一样的左右列表，左边为部门右边为部门对应的人员。这里可以看到某某部门的某某人员安排了什么岗位，如果未通知，我们可以对他进行短信通知。可对单个人员短信通知，也可以点击顶部的蓝色按钮对整个部门发起通知。

如果部门人数过多可以对姓名进行模糊搜索，我们也可以筛选短信已发送跟未发送的人员，这里的编辑按钮是跳转到【排班编辑】页面。

# 6. 更多

点击【左上角】可以返回首页，内容分为应用版本和开发公司相关信息。
