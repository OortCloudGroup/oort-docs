通知公告 使用手册

# 开始使用

  • 1.应用首页
  • 2.新建通知
  • 3.详情
  • 4.更多

# 1.应用首页

进入首页，顶部栏左侧是应用标题，右侧图标依次是【更多】以及【退出应用】。 首页内容分为收到和发布通知两个列表，每个公告上都显示了该公告的标题、内容和发布人和发布时间等信息，如果收到人为未读状态该公告左上角将会有个红点进行标记。

# 2.新建通知

右下角有个加号按钮点击进入新建通知页面，新建通知分为普通和活动两种类型，可点击顶部按钮进行切换类型。

# 3.详情

进入通知详情页可将当前通知标记为未读状态。

如果通知是你发布的可选择再发一条或撤销发布。

# 4.更多

点击【左上角】可以返回首页，内容分为应用版本和开发公司相关信息。
